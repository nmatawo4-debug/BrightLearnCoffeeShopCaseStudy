# BrightLearnCoffeeShopCaseStudy

The analysis was conducted as part of a data analytics task to assist a newly appointed CEO in making data-driven decisions.

The project focuses on identifying:
                                    1. Top revenue-generating products
                                    2. Best-performing time periods
                                    3. Sales trends across product categories
                                    4. Opportunities to improve overall store performance
                              
The main objectives of this analysis were to:
                                             *Identify which products generate the highest revenue
                                             *Determine peak sales hours
                                             *Analyze seasonal sales trends
                                             *Detect low-performing product categories
                                             *Provide strategic recommendations to improve business performance

The Bright Coffee Shop Sales dataset contains daily transactional records including: (Transaction date, Transaction time, Product category, Product type, Quantity sold, Unit price, Store location). These variables were used to evaluate customer purchasing behavior and sales performance patterns.
This project was completed using:
                                 1. SQL (data querying and aggregation)
                                 2. Databricks (data processing and analysis)
                                 3. Excel (data visualization) (adjust depending on what you used)
                                 4. GitHub (project version control)

Key business insigts from the analysis : Coffee generated the most revenue.
                                         Revenue was the lowest across all stores in the evening.
                                         Sales were relatively low during the summer season to the winter season
                                         
Recommendations: Introduce evening promotional offers
                 Launch seasonal beverage campaigns during warmer months
                 Bundle low-performing products with beverages
                 Implement customer loyalty reward programs
                 Increase staffing during peak morning hours
                 Introduce staff incentive programs to improve sales performance
